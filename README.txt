Braintree Hosted Fields Integration

Architecture:
- This integration was done with a node/express app and uses ejs for templating.
- Your merchantId, publicKey, and privateKey can be filled in lib/gateway.js.

Running the app:
First make sure you're using node version 8.11.1 or greater. Then run the following commands:
	npm install
	npm start

The app should then be running at localhost:3000
